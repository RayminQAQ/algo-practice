# 刷題計畫

> 對象：資工背景、DS/演算法底子紮實、兩年後才進職場、每週約 2~3 小時、同時有實習。
> 核心原則：**深度 > 數量**。你不缺理論，缺的是把理論變成「看到題目就認得 pattern」的反射，這靠少量題目深挖，不靠衝量。

---

## 一、為什麼不要「一天一題」

一天一題對你會有兩個問題：

1. **時間對不上。** 每週 2~3 小時，一天一題等於每題只剩 20 分鐘，會逼你「趕快看解答」——這正好是學不到東西的模式。
2. **心理負擔來自「連續性」。** 一天一題本質是 streak，漏一天就有挫敗感，實習忙的時候必爆。

改用**「每週配額」而不是「每日打卡」**：以「這週解 3 題」為單位，哪幾天做完全自由。實習爆炸的那天可以完全不碰，只要週末補回配額就好。單位從「天」放大到「週」，罪惡感就消失了。

---

## 二、週節奏（每週 3 題，約 2.5 小時）

| | 內容 | 時間 |
|---|---|---|
| 新題 ×2 | 依 NeetCode roadmap 順序推進 | 各 ~45 分 |
| 複習 ×1 | 重做一題「之前卡過」的舊題（見第四節 spaced repetition） | ~30 分 |
| 週回顧 | 週末花 15 分翻過本週筆記，更新 progress.md | ~15 分 |

前兩週還沒有舊題可複習，就 3 題全新。累積起來後改成 2 新 + 1 複習。

一年後（約 50 週）你會走完整個 NeetCode 150，而你有兩年——節奏非常寬鬆，完全不需要焦慮。

---

## 三、每一題怎麼做才學得到東西（最重要）

這是整份計畫的核心。**同一題，用對方法解 1 題 > 用錯方法解 5 題。** 每題照這個迴圈走：

1. **先硬想 20~30 分鐘，不准看解答。**
   學習發生在「卡住掙扎」的時候，不是看懂解答的時候。想不出最佳解沒關係，先寫出任何能動的暴力解。
2. **卡死了，只看提示，不看完整解答。**
   先問自己：這題屬於哪個 pattern？該用什麼資料結構？NeetCode 影片可以只看他「講思路」的前半段，關掉再自己寫。
3. **看懂解答後，關掉它，從空白重寫一次。**
   這步最關鍵。「看懂」是假象，「不看能重寫出來」才是真的學會。寫不出來 = 還沒學會，再看再寫。
4. **抽出 pattern，不是背答案。**
   每題結束用一句話記下：「**什麼特徵 → 用什麼技巧**」。
   例：*「找陣列中兩數和 → 用 hash map 邊走邊查補數」*、*「找子字串/子陣列的最佳區間 → sliding window」*。
   你要建的是一張「特徵 → 技巧」的對照表，這才是面試時真正在用的東西。
5. **在檔案頂部註解寫下思路與複雜度**（範本已內建欄位），當作給未來的自己看的筆記。

> 額外一招（模擬面試）：動手寫 code 前，先用講的或寫的把做法講一遍。強迫你把想法講清楚，也正是面試現場要做的事。

## 四、Spaced Repetition（間隔複習）——真正把題目變成長期記憶

解過就忘是正常的。用間隔複習對抗遺忘：

- 每題解完，在 `progress.md` 標一個**掌握度**：`✅ 順利` / `🤔 有提示` / `❌ 看了解答`。
- 標 🤔 或 ❌ 的題目，**排進未來的「複習」slot**：約 **1 週後** 重做一次、**1 個月後** 再重做一次。
- 複習 = **重做**（從空白寫），不是重讀。能順利重寫出來，才畢業移出複習清單。

這就是為什麼週節奏裡固定留一個複習 slot。長期下來，你留下的是真的內化的題，不是「解過但早忘了」的清單。

## 五、題目怎麼選：跟著 NeetCode 150 roadmap 走

不要在 LeetCode 隨機亂點。**按主題成批做**（一個主題連做 5~9 題）才建得起 pattern，跳著做建不起來。NeetCode 150 已經幫你把主題由淺入深排好了，直接照 `neetcode/README.md` 的 18 個主題順序推進即可。

因為你 Easy 幾乎是送分，**每個主題的 Easy 快速掃過 1~2 題暖身，主力放在 Medium**，這是面試的主戰場。Hard 題（下面標 🔴）第一輪可以先跳過，第二輪再回來。

## 六、語言與 Codewars 的定位

- **主力用 Python 學演算法。** 語法短、寫得快，讓你專注在「想法」而不是跟語法搏鬥。
- **C++ 選擇性重寫。** 挑那些你覺得特別經典的題，事後用 C++ 再寫一遍練手感（面試若要求 C++ 會用到）。不用每題都雙寫，那會拖垮節奏。
- **Codewars 當「低壓力暖身/換口味」。** 它偏語法趣味、不是面試主軸。實習累到不想動腦的那天，做一個 7~8 kyu 的 kata 保持手感即可，也算數進當週配額。

---

## 七、一句話總結

> 每週 3 題、不綁星期幾；每題硬想 30 分再看提示、看懂後關掉重寫；記下「特徵→技巧」；卡過的題隔週、隔月各重做一次。照 NeetCode roadmap 主題順序走，一年走完，你有兩年。慢即是快。
